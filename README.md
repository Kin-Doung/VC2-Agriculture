# Full Stack App: React + Laravel

## Structure
- `/frontend` - React JS (Vite)
- `/backend` - Laravel API

## Setup

### Backend (Laravel)
```bash
cd backend
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
pip install flask
pip install pillow
pip install numpy
pip install opencv-python
pip install tensorflow
php artisan storage:link
```
### Frontend (React)

Edit
